# Example: Research Project

## Project

Run a structured research initiative from question framing to validated findings.

## Roles Involved

CEO, Director, Researcher, Executor, Verifier, Memory Manager.

## Task Breakdown

- Define research question and success criteria.
- Decompose evidence gathering tasks.
- Execute analysis and synthesize outputs.
- Verify methodological and factual consistency.

## Feedback Loops

- Horizontal: Researcher <-> Verifier on evidence quality.
- Vertical: Director -> CEO for strategy pivots.

## Attention Interactions

- Manager highlights confidence gaps and stale assumptions.
- User can redirect scope when evidence contradicts priors.
