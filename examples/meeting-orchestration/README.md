# Example: Meeting Orchestration

## Project

Coordinate a cross-functional planning meeting with pre-reads, agenda synthesis, and decision tracking.

## Roles Involved

Director, Researcher, Executor, Scrum Master, Verifier, Memory Manager.

## Task Breakdown

- Gather context and required participants.
- Draft agenda and decision objectives.
- Execute meeting operations and capture outcomes.
- Verify action items and owners.

## Feedback Loops

- Horizontal: Scrum Master <-> Executor on flow bottlenecks.
- Vertical: Director escalation for unresolved dependencies.

## Attention Interactions

- Attention model prioritizes unresolved agenda conflicts.
- Override path allows scope compression before start time.
