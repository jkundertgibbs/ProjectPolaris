# Example: Malta Trip

## Project

Plan and execute a multi-phase travel project with budget and verification constraints.

## Roles Involved

CEO, Director, Researcher, Executor, Verifier, Bean Counter.

## Task Breakdown

- Define mission constraints (budget, dates, priorities).
- Research itinerary options.
- Decompose booking and logistics tasks.
- Verify confirmations and contingencies.

## Feedback Loops

- Horizontal: Researcher <-> Verifier on itinerary quality.
- Vertical: Director -> CEO for major scope or budget changes.

## Attention Interactions

- Manager surfaces deadline-sensitive tasks.
- User confirms or redirects booking actions.
