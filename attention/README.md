# Attention

## Purpose

Define attention as the control substrate for preparing and confirming actions.

## Key Concepts

- Continuous attention inference.
- Inspectable reasoning and confidence.
- Explicit user override and redirection.
- Privacy-bounded signal handling.

## Relationship to Polaris Architecture

Attention mediates how Manager proposes actions on Boards and Projects.
