# Visual Language

## Purpose

Set baseline visual semantics for Polaris artifacts.

## Key Concepts

- Distinct visual treatment for project, task, and role states.
- Emphasis on inspectable status and authority markers.
- Minimal ambiguity between inferred and confirmed actions.

## Relationship to Polaris Architecture

Visual language reinforces governance and attention transparency.
