# Task Economy

## Purpose

Describe a governed market-like model for distributed task execution.

## Key Concepts

- Task exchange across managers.
- Negotiated cost, risk, and deadline terms.
- Accountability and verification requirements.

## Relationship to Polaris Architecture

Task economy depends on `governance/economic-layer.md` and manager delegation contracts.
